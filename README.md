**DjavaLauncher** is a high-performance, feature-rich modified client for **San Andreas Multiplayer (SA:MP)** on Android. Built with a focus on aesthetics and performance, it provides a modern experience for playing GTA: San Andreas online.

---

## 🌟 Key Features

### 📱 Modern Launcher
- **Material 3 UI:** Clean, modern, and intuitive interface following the latest Android design standards.
- **Auto-Downloader:** Integrated system to download and verify required game assets automatically.
- **Server Management:** Easily add, manage, and connect to your favorite SA:MP servers.
- **Flexible Settings:** Deep customization for both the launcher and the game client via a user-friendly interface.

### 🎮 High-Performance Client
- **Optimized Graphics:**
    - 🏎️ **OpenGL Reflections:** Realistic real-time reflections on vehicles.
    - 🌊 **WaterShader:** Enhanced water rendering for a more immersive environment.
    - ☁️ **Custom SkyBox:** Beautiful, high-quality sky textures.
    - ⚡ **Performance Tweaks:** Disabled unnecessary elements like `CCorona` for maximum FPS.
- **Enhanced Gameplay:**
    - 🎙️ **Voice Chat:** Built-in high-quality voice communication using the Opus codec.
    - 🛠️ **ImGui UI:** Professional in-game menus, chat box, and scoreboard using the ImGui framework.
    - 📱 **Multi-Touch Support:** Improved controls specifically tuned for mobile devices.
    - ⌨️ **Custom Keyboard:** Optimized input for chatting and commands.
    - 🥊 **Kick Button:** Native support for the kick action in-game.

---

## 🛠️ Technical Specifications

| Feature | Description |
| :--- | :--- |
| **Networking** | RakNet-based synchronization for low-latency gameplay. |
| **Native Core** | Developed using C++20 for peak performance and efficiency. |
| **Hooking** | Advanced function intercepting using `Shadowhook`. |
| **Security** | Protected with `paranoid` (Java) and `obfusheader.h` (Native). |
| **Audio** | Powered by the BASS library for high-fidelity sound. |
| **Compatibility** | Fully compatible with SA:MP 0.3.7 servers. |

## 🚀 Getting Started

### Prerequisites
- Android device running Android 8.0 (Oreo) or higher.
- A legitimate copy of Grand Theft Auto: San Andreas (v2.00 or higher recommended).

### Installation
1. Download the latest `app-debug.apk` from the [Releases](https://github.com/Nathan/DjavaLauncher/releases) page.
2. Install the APK on your Android device.
3. Open the launcher and follow the prompts to download the necessary game assets.
4. Set your nickname in the settings.
5. Head over to the server list and start playing!
